When a colon is used in combination with an equality-like operator (such as :=), the colon may appear slightly vertically misaligned with the equality sign bars. The mathtools package provides replacement commands for a nicer visual result.

Note: The mathtools package also provides a magical way to automatically use the right colon, by using \mathtoolsset{centercolon}. This removes the need of using the replacement commands. It is however not recommended, as this setting is not compatible with some babel languages (such as French). For more details, see section 3.7.2 of the mathtools documentation.

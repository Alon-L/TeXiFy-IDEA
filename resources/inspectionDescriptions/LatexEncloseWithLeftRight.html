Whenever commands are used in math mode that alter the default line height, \leftX..\rightX must be used instead of regular brackets to make sure the brackets have the right height.

This inspection checks the following commands: \frac, \dfrac, \sqrt, \sum, \int, \iint, \iiint, \iiiint, \prod, \bigcup, \bigcap, \bigsqcup, \bigsqcap

A functionality to add your own commands is coming somewhere in the future (or earlier if you want to implement it yourself: just send us a pull request!).
